# ناميكس | Namix
## بروتوكول إدارة الأصول الرقمية المتقدم

ناميكس هي منصة استثمارية رائدة مصممة لإدارة الأصول الرقمية والتداول بالذكاء الاصطناعي، توفر تجربة مستخدم نخبوية مع حماية قصوى لرأس المال.

### 🚀 المواصفات التقنية
- **الإطار البرمجي:** [Next.js 15 (App Router)](https://nextjs.org/)
- **قاعدة البيانات:** [Firebase Firestore](https://firebase.google.com/)
- **الهوية البصرية:** Tailwind CSS + Shadcn UI
- **محرك الذكاء الاصطناعي:** Google Genkit
- **المزامنة:** Binance WebSocket API

### 🛠️ ميزات المنصة
- **مختبر العقود (Contract Lab):** هندسة عوائد مؤتمتة بالكامل عبر بروتوكولات استثمارية ذكية.
- **التداول الفوري (Spot Trading):** محرك تداول لحظي متصل بأسواق بينانس العالمية.
- **ساحة المغامرة (Arena):** تجربة ترفيهية استراتيجية تعتمد على احتمالات السيولة لنمو إضافي.
- **مركز السفراء:** نظام إحالات وعمولات متطور لنمو الشبكة الاحترافية.
- **لوحة التحكم الإدارية:** قمرة قيادة كاملة للمشرف للتحكم في كافة مفاصل النظام والسيولة.

### 💻 البدء بالتشغيل
لتشغيل المشروع محلياً في بيئة التطوير:

```bash
npm install
npm run dev
```

افتح [http://localhost:9002](http://localhost:9002) في متصفحك لمعاينة المنصة.

### 📁 هيكلية المجلدات
- `src/app`: تحتوي على كافة الصفحات والمسارات (Routes) وبروتوكولات الـ SEO.
- `src/components`: المكونات البرمجية القابلة لإعادة الاستخدام والواجهات التفاعلية.
- `src/firebase`: إعدادات وخدمات الاتصال بقاعدة البيانات والتوثيق اللحظي.
- `src/lib`: المكتبات والمحركات المساعدة (AI Engine, Technical Indicators).
- `public`: الملفات العامة وصور المشاركة (OG Images).
