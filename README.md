# ناميكس برو | Namix Pro
## بروتوكول إدارة الأصول الرقمية السيادي

ناميكس برو هي منصة استثمارية متطورة مصممة لإدارة الأصول الرقمية والتداول بالذكاء الاصطناعي، توفر تجربة مستخدم نخبوية مع حماية قصوى لرأس المال.

### 🚀 المواصفات التقنية
- **الإطار البرمجي:** [Next.js 15 (App Router)](https://nextjs.org/)
- **قاعدة البيانات:** [Firebase Firestore](https://firebase.google.com/)
- **الهوية البصرية:** Tailwind CSS + Shadcn UI
- **محرك الذكاء الاصطناعي:** Google Genkit
- **المزامنة:** Binance WebSocket API

### 🛠️ ميزات المنصة
- **مختبر العقود (Contract Lab):** هندسة عوائد مؤتمتة بالكامل.
- **التداول الفوري (Spot Trading):** محرك تداول لحظي متصل بأسواق بينانس.
- **ساحة المغامرة (Arena):** تجربة ترفيهية استراتيجية لتحقيق نمو إضافي.
- **مركز السفراء:** نظام إحالات وعمولات متطور لنمو الشبكة.
- **لوحة التحكم الإدارية:** قمرة قيادة كاملة للمشرف للتحكم في كافة مفاصل النظام.

### 💻 البدء بالتشغيل
لتشغيل المشروع محلياً في بيئة التطوير:

```bash
npm install
npm run dev
```

افتح [http://localhost:9002](http://localhost:9002) في متصفحك لمعاينة المنصة.

### 📁 هيكلية المجلدات
- `src/app`: تحتوي على كافة الصفحات والمسارات (Routes).
- `src/components`: المكونات البرمجية القابلة لإعادة الاستخدام.
- `src/firebase`: إعدادات وخدمات الاتصال بقاعدة البيانات.
- `src/lib`: المكتبات والمحركات المساعدة (AI Engine, Technical Indicators).
- `public`: الملفات العامة وصور المشاركة (OG Images).
